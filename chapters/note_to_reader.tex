\section*{Document Conventions}
Links are shown in:
\begin{itemize}
    \item \textcolor{blue}{blue} for links to other parts within this book, such as chapters, sections, glossary, and page numbers.
    \item \textcolor{red}{red} for links to the reference section.
    \item \textcolor{orange}{orange} for links outside the book, such as a website.
\end{itemize}

Code and shell commands are shown in:
\begin{itemize}
    \item Shell commands (zsh, bash) are shown inside a box outlined in black, with a note above it that identifies it as shell commands.
    \item Python commands are shown inside a box outlined in black, with a note above it that identifies it as code.
\end{itemize}
